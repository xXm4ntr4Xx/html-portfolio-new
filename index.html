About me

I'm 33 years old guy from Italy. After working 9 years in a casino as a croupier I decided to challenge myself by joining a different sector. I have always been interested in coding, I have always watched tutorials on youtube and I was always amazed at what people can produce by writing a simple line of code. During the journey at 'School of code' we deal with different topics : We learned the difference between UI and UX. We learned how to pair programming, how to break down a problem, use computational thinking and apply agile methodology to improve teamwork, and how to reach an MVP(most valuable product) to deliver. We started with the basics of HTML and CSS and after we move to Javascript to make the website interactive. After learning the basics of javascript we start to implement some frameworks to make development easier like (React). We learned how to create a basic server with node js using a framework like express.js and create a simple CRUD application(create, read, update, delete functionality). We learned how to work with databases like Heroku and how to connect databases with our express server and using the query string to use different actions on the database element. We learned different types of testing (unit tests like Jest and superset), mock test, integration test(cypress). We learned about Typescript(a superset of javascript) the makes the code more robust and solves most of the weird behavior of Javascript. We learned about docker and how to use a container to develop a project in different environments.
